## Einführung in JavaScript

* JavaScript wird häufig zusammen mit Begriffen wie Ajax (moderner: fetch oder Axio), 
    * Angular, React oder allgemeinen Webtechnologien genannt.
* JavaScript bildet die Grundlage für alles, was im Browser dynamisch ablaufen soll.
* HTML liefert den Inhalt, CSS das Aussehen — **JavaScript steuert Verhalten und Interaktion**.
* mit JS können `click-Events` auf Html-Elemente definiert werden
* Mit JavaScript können Inhalte, `Attribute` oder `Styles` im DOM verändert werden.
* oft werden Klassen durch JS verändert
    * z. Bsp. `is-open` kann dynmisch durch ein Event hinzugefügt werden
        * diese Klasse kann bereits in CSS Styles definieren
* Auch Formularvalidierung ist ein zentraler Anwendungsfall.
* Viele moderne Browserfunktionen sind über **Web-APIs** erreichbar (z. B. Storage, Audio, Video, Netzwerk).
* Eine API ist eine Schnittstelle, über die Programmfunktionen bereitgestellt werden.
* JavaScript ist heute nicht mehr nur clientseitig: 
    - es wird auch für Server, Desktop-Apps, mobile Apps, Spiele, 3D-Grafik und Embedded-Geräte genutzt.

**Interpretierte Ausführung**

* JavaScript ist eine interpretierte Programmiersprache.
* Ein Interpreter führt den Quelltext direkt aus und übersetzt ihn Schritt für Schritt in Maschinencode.
* Der Interpreter ist Teil des Webbrowsers.
* Der Code kann mit jedem Texteditor geschrieben und sofort ausgeführt werden.
* Die Ausführung ist tendenziell langsamer als bei kompilierten Sprachen.

**Kompilierte Programmiersprachen**

* Kompilierte Sprachen werden vor der Ausführung vollständig in Maschinencode übersetzt.
* Das Ergebnis ist ein eigenständiges, ausführbares Programm.
* Beispiele: C, C++, Swift.

**Zugriff auf das HTML-Dokument**

* Reaktionen auf Benutzereingaben (z. B. Klicks) sind möglich.
* Inhalte und Darstellung im Dokument können dynamisch verändert werden.
* Die Änderungen betreffen nur das Dokument im Arbeitsspeicher – nicht die HTML-Datei auf dem Server.
* Diese Interaktion wird über das Document Object Model (DOM) ermöglicht.

**Ziel von JavaScript in der Webentwicklung**

* Nutzerinteraktionen verarbeiten.
* Dynamische Anpassungen vornehmen.
* Die Bedienbarkeit und das Verhalten einer Website verbessern.

**Zusammenspiel der drei Webtechniken**

* HTML = Inhaltsschicht.
* CSS = Darstellungsschicht.
* JavaScript = Verhaltensschicht.
* In der Praxis werden oft alle drei kombiniert.
* HTML ist zwingend nötig, CSS und JavaScript optional, aber in der modernen Praxis üblich.

**Grundidee von Variablen**

* Variablen werden verwendet, um Werte zu speichern und weiterzuverarbeiten.
* Typische Quellen für solche Werte: Benutzereingaben, Formularfelder, Datenbankabfragen.

* Eine Variable verweist auf eine Speicheradresse im Arbeitsspeicher.
* Der JavaScript-Interpreter kann während der Programmausführung auf diesen Wert zugreifen.

* Eine Variable kann **jeden beliebigen Typ** aufnehmen:

      * Zeichenkette (String)
      * Zahlen
      * Arrays

* Variablen können ihren Typ zur Laufzeit ändern (z. B. erst Zahl, später String).
* allerdings sollten Programme so gestaltet werden, dass der Typ sich nicht ändert (minimier Fehlerwahrscheinlichkeit)

**Deklaration von Variablen**

* Variablen werden mit `let` (oder `var`) deklariert
* Werte können direkt bei der Deklaration mit `=` zugewiesen werden.

```js
let breite = 5
let einText = "Meldung"
```

* `let` sollte bevorzugt werden.
* `var` ist möglich, aber veraltet

**Regeln für Variablennamen**

* Müssen mit einem Buchstaben oder `_`beginnen.
* Keine Leerzeichen, Sonderzeichen oder Umlaute.
* JavaScript-Schlüsselwörter sind nicht zulässig.
* Groß- und Kleinschreibung unterscheiden sich (case-sensitive):
    * `var01` und `Var01` sind zwei verschiedene Variablen.

* Beispiele reservierter Wörter:
  `async`, `await`, `break`, `case`, `class`, `catch`,
  `const`, `continue`, `debugger`, `default`, `delete`, `do`,
  `else`, `enum`, `export`, `extends`, `finally`, `for`,
  `function`, `if`, `implements`, `import`, `in`, `interface`,
  `instanceof`, `let`, `new`, `package`, `private`, `protected`,
  `public`, `return`, `static`, `super`, `switch`, `this`,
  `throw`, `try`, `typeof`, `var`, `void`, `while`,
  `with`, `yield`.

**Definition von Konstanten**

**Definition von Konstanten**

* Konstanten werden in JavaScript mit dem Schlüsselwort `const` definiert.
* Sie müssen bei der Deklaration **sofort initialisiert** werden.
* Der Wert einer Konstante kann nach der Zuweisung nicht mehr geändert werden.

**Beispiel**

```javascript
const TVAL = 'Testausgabe';   // Konstante anlegen
console.log(TVAL);            // Ausgabe: Testausgabe

TVAL = 'Neue Testausgabe';    // Fehler: Wertänderung nicht erlaubt
console.log(TVAL);
```

### **Boolean-Datentyp**

### **Datentypen konvertieren**
* JavaScript konvertiert Typen dynamisch.
* Beispiele:

  ```javascript
  let val = 123;       // number
  val = "Jetzt Text";  // string
  ```
* `+` mit String → Ergebnis ist **String**.

  ```javascript
  "100" + 44  // "10044"
  ```
* `-`, `*`, `/` erzwingen Zahl-Konvertierung:

  ```javascript
  "100" - 44 // 56
  "10" / "2" // 5
  ```
* Für explizite Konvertierung:

  * `parseInt()` → Ganzzahl
* `parseFloat()` → Fließkommazahl 
* Beispiel:

  ```javascript
  let iVal = parseInt("100") + 44; // 144
  ```

**JavaScript-Funktionen**

* Eine **Funktion** ist ein **Block Code**, den man **mehrmals ausführen** kann.

### 1) Funktion definieren

```js
function greet() {
  console.log("Hallo!");
}
```

### 2) Funktion ausführen (aufrufen)

```js
greet();   // gibt "Hallo!" aus
```

### 3) Funktion mit Parameter

Parameter = wie eine Eingabe.

```js

function hello(name) {
    console.log("Hi " + name)
}

hello('Bob')
```

### 4) Funktion mit Rückgabewert

```js

function f(x) {
    return 2 * x
}

f(5) // 10


function add(x, y) {
    return x + y
}

```

# Jquery

* jQuery vereinfacht komplexe JavaScript-Aufgaben und automatisiert gängige Interaktionen.
* Die Bibliothek ist ideal für Webentwickler, die mehr HTML/CSS- als JavaScript-Erfahrung haben.
* Einsteiger profitieren vom einfachen Einstieg.


```js
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>
    <button id="btn">Klick mich</button>
    <p id="msg">Hello World</p>
    
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
    <script>
        $("#btn").on("click", function () {
            console.log("geklickt")
            $("#msg").text("Hallo jquery!")
        })
    </script>
</body>
</html>
```
**selbe Bsp. ohne jquery**
```html

</head>
```
