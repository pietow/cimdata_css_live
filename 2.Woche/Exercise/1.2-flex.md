# **Übung: Flex-Basis und Flex-Grow verstehen**

## **Ausgangscode**


```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        * {
            box-sizing: border-box;
        }
        .container {
            border: 1px solid red;
            display: flex;
            width: 1000px;
        }
        .item {
            padding: 2em;
            border: black 1px solid;
            flex-basis: 20%;
        }
        .item2 {
            /* flex-grow: 1; */
        }

    </style>
</head>
<body>
    <div class="container" >
        <div class="item item1">20% Box</div>
        <div class="item item2">20% Box</div>
        <div class="item item3">20% Box</div>
    </div>
</body>
</html>
```

---

# **Aufgabe: Was passiert mit den Boxen?**

### **Schritt 1 — Beobachte den Ausgangszustand**

Fragen an die Schüler:

1. Wie breit ist die `.container` Box?
2. Wie breit ist jede `.item` Box, wenn `flex-basis: 20%` gesetzt ist?
3. Wieviel Platz bleibt übrig?

---

## **Schritt 2 — Starte das Experiment**

Die Schüler sollen folgendes tun:

### **A) Entferne den Kommentar bei `flex-grow: 1;`**

```css
.item2 {
    flex-grow: 1;
}
```

### **Frage 1:**

Was passiert jetzt mit der **zweiten Box**?

### **Frage 2:**

Warum verändern sich die erste und dritte Box **nicht**?

---

## **Schritt 3 — Mehrere Grow-Werte testen**

### **A) Füge flex-grow zu allen Items hinzu:**

```css
.item1 { flex-grow: 1; }
.item2 { flex-grow: 1; }
.item3 { flex-grow: 1; }
```

### **Frage 3:**

Wie wird der freie Platz verteilt?


---

## **Schritt 4 — Unterschiedliche Wachstumsraten**

### **A) Setze:**

```css
.item1 { flex-grow: 1; }
.item2 { flex-grow: 2; }
.item3 { flex-grow: 1; }
```

### **Frage 4:**

Welches Item erhält jetzt am meisten Platz – und warum?
